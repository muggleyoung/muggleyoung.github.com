学习LaTeX-1

段首缩进：

默认的是首段不缩进，后面的才会有。加上usepackage{indentfirst}就会每段都缩。

还有一个，是我自己一开始就没有注意到的。在LaTeX中，回车并不作为新一段的开始，需要空一行才能识别。

 

关于项目编号：(numerate编号)

假如连续几个编号之间隔了几段正文，或者说，想要从一个非1的编号开始。如：

\begin{enumerate}
\item  AAA
\end{enumerate}

XXX

\begin{enumerate}
\item BBB
\end{enumerate}

可以这样

\begin{enumerate}
\item  AAA
\end{enumerate}

XXX

\begin{enumerate} \stepcounter{enumi}
\item BBB
\end{enumerate}

这样，BBB的编号就是2。假如需要编号为3，4，5 。。。

\begin{enumerate}
\item AAA \item AAA
\end{enumerate}
XXX
\begin{enumerate}\stepcounter{enumi}\stepcounter{enumi}
\item BBB
\end{enumerate}

这样的话，BBB的编号就是从3开始。明白没？也就是说从n开始编号，\begin{enumerate}后面就要写n-1个\stepcounter{enumi}。
