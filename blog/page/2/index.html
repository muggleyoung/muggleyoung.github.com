给亲爱的王不拔

Jun 8th, 2011

亲爱的王不拔小姐：

    签证被拒那天我心情也很差，所以才会想寄软香酥给你。这确实是一个没那么好的机会，但我一直认为you deserve it，好比你那200美刀的奖金。即使是对你3年工作的安慰奖，那也总能让人心理稍稍舒坦些。

    对我来说，研究生3年实在过的不满意。那天跟小冯同学说起这事儿，我说到了这3年过得多么没有意义，不快乐，小冯说这就是你最大的收获，知道自己根本不想要这样的生活。我一下释然了。

    作为新鲜人，或许我表达了太多对于工作的兴奋。常有同事问，工作了感觉怎么样。我都干脆的回答，比上学时舒坦多了。可不是么，找到一个无比欢乐的工作环境，喜欢的事情，觉得生活变得有意义，充实起来，我打心眼里的开心舒坦，恨不得见人就说我有多喜欢这份工作。想想你充实工作的3年，我是怎么熬过来的，心里会平衡些吗。

    其实你不必被我的状态影响，每个人处于每个阶段都有不同的快乐点。现在每天去健身房不也是很快乐的事情么。

    关于北京，我在这里出差，什么都不用操心，自然过得滋润。即使薪水两倍于我的同学，在北京生活的质量和尊严感也没法和我比，而积攒的银子并没有比我多出多少。你该明白我的意思吧。至少我现在不担心房租，住在二环边上，上班走走就到，偶尔向你们抱怨下吃的不如家里好吃，但也没亏着自己。可是你一个人来北京生活，那就不一样了。房租很贵，即使这样还有可能要坐半小时以上的地铁去上班。我目前能看到的结果，要么经济不大宽裕，要么生活质量无法满足我们这类娃的要求。

     鉴于我对杭州极其极其不稀罕，所以北京可以算是一个just so so的选择。那天的聊天，只是想告诉你生活两个月后我对这个地方产生的一些顾虑。你来了，我们俩会有更多机会海吃闲逛八卦呢。所以放心吧，你的决定我自然会尽力帮忙。

     我们是生活在现实中的理想主义者吧。我从来没有为你的大龄剩女问题担心过。我们没有过多的要求，不会为喜欢的人加上经济条件的限制，只是想找到一个让我们感到自在开心的娃而已，对吧。

实在想不起你还写了什么给我，白天攒了一肚子的话，到这时也变得没有逻辑没有条理。总之吧，我喜欢现在有追求充实的生活状态，我希望你尽快从现在的坏状态中解脱，所以有什么需要帮忙的，尽管开口呗。你的闺蜜只是刀子嘴而已。

                                                                                 你亲爱的杨也尼，葛郎台

牛豆豆同学教育我：师父教东西的时候不要偷懒，记在本子或者txt上

第一次跟钱钱pair搭环境的时候，光忙着用眼睛看

咱有blog的娃，咋能光往本儿上记事儿呢

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

话说这搭环境啊

mini项目环境也简单

可出点问题就能晕乎乎的折腾好久，比如说某条命令多了一个空格。。。。

 

一个一个说吧～

tomcat：mac下tomcat安装，貌似有dmg文件，也可以从tomcat主页下载zip包。

安装或解压好之后，vi ~/.profile

set tomcat home:后面的目录就是tomcat所在的位置

export TOMCAT_HOME = /Users/twer/workspace/apache-tomcat-7.0.12

一般启动tomcat都要进入bin目录输入startup.sh，钱钱教我可以设置别名

start_tomcat:

alias start_tomcat = $TOMCAT_HOME/bin/startup.sh

stop_tomcat:

alias stop_tomcat = $TOMCAT_HOME/bin/shutdown.sh

这样一来，可以在任何目录下输入start_tomcat, stop_tomcat启动和停止服务器。省事很多

 

ant：

安装ant，在~/.profile中加入：

export ANT_HOME = /Users/twer/workspace/apache-ant-1.8.2

 

git：

安装git，使用git clone 命令clone一个reporsitory，git pull更新代码

 

IntelliJ IDEA：

安装之后，选择project struction->project->project SDK->new->jdk，找到：system-library-java-javaVirtualMachine-1.6.0.jdk

打开自己的工程就ok

哦，好像还得在profile里设置下project home

 

mysql：

安装没啥说的，设置路径

export PATH = /usr/local/mysql-5.5.11-osx10.5-x86/bin$PATH

导入数据库：mysql -u root targetDatabase < sourceDatabase.sql

导出命令用mysqldump，箭头方向不同而已

 

操作数据库时，我们使用了sequel pro，安装设置好连接参数就行

 

然后！start_tomcat之后，咱的程序就run起来啦！

帮助文档有帮助

May 4th, 2011

上班大约10天时，项目临近上线，开发忙着完成stroy，我这个二把刀测试，除了尽量全面的测试，支持他们的工作，还开始着手写项目帮助文档。

写文档的过程总是痛苦又无聊的。

需要考虑怎样组织逻辑的条理性，包括整个文档结构和你的每一句话；

怎样让用户可以更快捷的找到他们需要的操作；

怎样直观的说明问题，而不是一堆让自己看了都昏昏欲睡的文字；

很多我们看来简单的不能再简单的操作，也要仔细交代；

基于上一条，但有些基础的介绍，几百年都不会有人看，可是万一有不会的，文档里面找不到，就是我们考虑不周到，所以唠唠叨叨说了很多之后，怎样突出操作中关键的逻辑，这也是个问题。

 

幸好咱是刚写过论文的孩只，憋几个文档还是可以忍受的。

近4天时间，还狠变态的花了时间整理字体、排版等面子工程。

当时觉得，这几个文档，算不上好的，也算是above average了。

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

项目上线后，一直参与着surprot的工作，突然，就意识到的帮助文档的重要性。

有一类用户，遇到问题会逐步向你说明出现问题时的操作，还体贴的辅以截图。当然我不会一眼看出问题，两眼也看不出，但我经验丰富的pair会。他的经验，加上我测试时对系统的了解，定位问题还是比较迅速的。

另一类用户，由于不了解系统规则，不当操作之后，出现错误，没有预期的结果，他们会莫名其妙，描述问题时，我们也听的狠晕。这时候就多了自己重现问题的工作量。

还经常会出现的问题是，研究了很久，发现系统就是这样的逻辑，只是用户不了解，而我，也不是狠了解。

除了反省自己测试覆盖还不够，对系统需求了解不够多以外，我们是不是需要向用户介绍更多，至少成为这个系统的半个专家。

介绍系统实现的功能，用户操作的基本限制，还要尽量多的说明一些操作上的逻辑－业务逻辑，系统实现上的限制等等。这样可以减少第二类用户出现问题的频率，用户在出错时，可以在帮助文档中找到解决方案，或知道哪里出问题了，交流起来也会更高效。

还有一种方案，就是把具有管理员权限的用户培养成半个专家。其他用户出问题，大多通过求助管理员解决。我们也可以更专注于真正的优化系统，维护完善功能的工作上。

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

用户越是了解系统，我们的维护和后续的开发就会越顺利。

想起刚开始写文档时，愁得不知道该写啥，啥该写

钱钱问我，如果你是用户，你最需要了解什么

我迷惑了一中午。

surport之后，总算有了点体会

其实这个嘛，也可以BA和QA pair完成～嘿嘿～

有家房屋主人门口写着：狗不怕你，你怕狗么？此话反倒吸引路人忍不住爬上墙偷看是何等恶犬。。。

鼓浪屿的景点有联票，比较划算

我不爱逛景点，只选择了钢琴博物馆和风琴博物馆两家，门票大约或者貌似20 or 30

这俩博物馆还是值得瞅的，教训就是逛博物馆前最好做功课

像咱这种木有音乐细泡的娃，着实搞不懂面前的俩钢琴区别在哪儿，有啥优劣。。。。

鼓浪屿还有音乐厅，几乎每晚都有演出，偶尔收费。7点钟去音乐厅就好

人品不好，那天下雨，因为天气原因，乐队没来。。。。

 

没有相机，没有照片，回忆起来真的有困难，都是片段，没法串起来

讨厌的小偷啊。。。。。

 

下午会厦门岛内，逛厦大

依旧人品负爆发，雨下的不停，进了厦大，越下越大

北方娃对潮湿的气候实在不习惯，浑身不得劲，也没仔细逛

结合之后去集美观察结果，厦大还是很有韵味的，毕竟时间久远些。建筑风格还是类似的，南洋风格

厦大西门旁边就是南普陀，特别有爱的地儿

寺门口的两座塔上蹲满了各种鸟，鸽子居多

水池里各种鱼，青蛙，乌龟泛滥。说实话，这个看着有点儿慎得慌。。。。想起前两天在新校区孔哥抓到的蛤蟆。。。。

南普陀门票不贵，3块还是5块，但游客太多，看着头疼

我不是虔诚教徒，没进去

旁边有素斋馆，似乎口碑不错，还有南普陀的素饼

恩，我查攻略净看好吃的了。。。。

绕厦大沙滩走了很远，湿乎乎的天气实在不舒坦

晚上吃了海边大排档，不算便宜，味道还行，我对海鲜的热爱。。。那是相当的。。。。

晚上早早休息，一为早点摆脱身上的潮衣服；二为第二天去集美攒劲儿

事实证明，这个决定很明智

我不出所料的迷路了～

毕业旅行-厦门1

Mar 29th, 2011

3月18日，下午

收拾一书包行李，出发

西安少有的春天，暖和的天气，心情亦好

晚上到厦门，刚刚下过一场小雨

空气清新湿润，激动入住青旅

PS：机场跟着指示牌走，可以找到公交站

我住的南华路厦门国际青年旅舍，会员45一晚/床位，坐车去火车站转车，到理工学院即可

 

3月19日

一夜的雨，早起依然

看着地图，打算向西北方向走去轮渡

早饭随便找了小吃店，点没听过的沙茶扁食

味道像芝麻酱馄饨

    恩，沙茶酱主成分有花生芝麻之类，扁食，基本就是我们吃的馄饨

走着走着，就到了东北方向的民族路

乌塘沙茶面，排着长队

地图上有推荐，怀着游客精神忍受排队品尝

海蛎面加米血鸭血，比之前小店的好吃，价格也不贵，10块左右

走路至轮渡，与在此工作同学碰头，开始鼓浪屿之行

 

鼓浪屿：

一直下雨，浑身潮湿

绕岛一整周，不算迷路的

中间的各种路也走了不下3遍

建筑很有南洋特色，风景好看，可惜下雨有雾

鼓浪屿上好吃的很多

海蛎煎：最好吃的是龙头路183号（门牌不一定准确哈），10块一份，很多店里要15的。后来尝试过对面家的，没有这家细致；

鱼丸：原巷口鱼丸，尝试鱼丸，鱼羹，炸五香，煎饼，土笋冻。鱼丸确实不错，汤鲜，我这平时不爱吃鱼丸的评价呢~也尝试了别家鱼丸，没有这家筋道，大约是蛋清之类放少了；

马拉桑：龙头路上有很多马拉桑，同学带我去的俩大妈经营的小店，一人宽，不记得几号了。一天喝3杯，真好喝，而且便宜。想当初在西安点鲜榨果汁，20多块，还经常有掺水的。。。。

奶茶：攻略里都会提到的张三疯奶茶，太甜，腻嗓子，但是奶茶里加了葡萄干还有类似玉米麦片的东西，也因为这样，大概才会那么甜的hou人。还有一家古追奶茶，更和我口味，茶味比较重。这家奶茶店最好早一点去买，每天外卖的数量有限，非外卖的会贵一些。两家奶茶都是十几块一杯的。

风景，下次讲～
