西安海泰笔试

抓紧时间记录下昨天的笔试

西安海泰公司，有天宣讲请实验室同学代投的简历，到了才知道投的是软件开发，可还是不知道哪个部门了。

题目很广很广，啥都有，操作系统，数据库，网络，C++，还有画频谱，讲数据采集过程之类的。完全不会的。

事实证明，后来很多人说要换系统的试卷，拿到手看了一眼，又统统举手要求换回软件了，哈哈哈，不够麻烦的。

具体的题目不记得几个了，主要是范围很广，但说不上有多么难，并且选择挺多的，似乎有几个填空空的。

最后的大题是选做题，七选二，很扯的是，七个里面我只会两个，没的选啊。

一个写代码的，冒泡排序

还有一个讲TCP/IP的区别，优缺点，还有画时序图。不知道时序图是啥，就花了TCP的握手协议，和自己心中理解的UDP通信状况。

长假碎碎念-la Vie en Rose

虽然没啥心情，想了想，这么长的假期，这么多的碎事，还是絮叨下吧

这一絮叨，就得从上月29号开始

晚上崔同学勾引我出去玩，结果还没吃完饭，娃就因为感冒挂掉，早早就各回各家了。

BubbleSort

冒泡排序：

#include <fstream>
#include <math.h>
using namespace std;

void BubbleSort(int A[],int n)
{
    for(int j=1;j!=n;j++)
    {
        for (int i=0;i!=(n-j);i++)
        {
            if(A[i]>A[i+1])
            {
                int t = A[i];
                A[i] = A[i+1];
                A[i+1] = t;
            }
        }
    }
}

void main()
{
    ifstream infile;
    ofstream outfile(“result.txt”);
    infile.open(“seq.txt”);
    int A[100];
    int n=0;
    while(!infile.eof())
    {
        infile>>A[n];
        n++;
    }
    BubbleSort(A,n);
    for (int i=0;i!=n;i++)
    {
        outfile<<<” “;
    }
}

网易笔试

网易2011年校园招聘笔试，跟有道和游戏是分开进行的，另外那俩我还没投简历呢~

传说中的笔试题难的bt

我投的测试工程师，一如既往的BT

四部分：计算机基础，Java，数据库，测试

 

计算机基础：没几个题，记得有算法复杂度，tcp三次握手，最后还有一个写代码的题目，我是深深深深的记住了

    整形数组a0,a1,….,an，找出数组中的最大连续子串和，即ax,a(x+1),….,ay的和最大，要求算法复杂度为O(n)，写出思路和代码，不限语言。

   这题纠结了好久，还是把自己绕进去了。。。。。

Java：选择题，读代码题。基本围绕面向对象进行的。类类型，类继承之类。Java我不会，凭印象做的

数据库：忘完了，直接放弃。考了索引类型，写spl语句检索之类

测试：黑盒白盒，测试用例的概念分类什么的；qq能登陆，ie却打不开网页，问你会从哪几个方面解决问题；生产出的一次性水杯，设计测试用例；写一段代码，什么代码忘了，让自己测试该段代码。

 

不同部门不同职位，题目都是不一样的。北京的测试，全考的测试知识；开发，还有智力题要考之类之类。。。

接连几次笔试被鄙视，太太沮丧了，好好看书了我，不投简历先~
