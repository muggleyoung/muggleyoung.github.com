再游北京-3

4月18日，3rd day：

终于奔798去了。

上次来北京，就没去成798吧，让我遗憾了好几年。当然，这次还有一地儿最后没去成，这下又有的遗憾了。。。嘿嘿，别说我是故意的。。。

可怜了WD同学，周末的大早上还要上闹钟，磨磨唧唧出门9点多

中间先去大剧院兑奖。本以为可以换到当晚的票，看完次日正好回家，可是我中的是二等，说二等的票都安排在了19号晚上，无奈了。我一个劲的说我时间不方便，那些工作人员跟没听见似的，光忙着恭喜我。。。太虚伪了么

没辙，看着660快的票价，实在舍不得把两张票都留给WD，不是我小气，是我也想去看呐。。。。

我俩考察了下，现场倒卖也不太现实。再说了，总价1k多的门票，你说我倒卖的话，卖多钱合适呢。太复杂了。

最简单的解决方案，就是，我去退了火车票，买晚一天的回。

好吧，弄好兑奖这事儿就可以吃午饭了。我俩硬忍着，先坐车到了798门口。话说这798和WD家显然就是穿越了整个北京城么。。。。

饿的不行，附近没啥吃的。找了家蛋糕店，金凤呈祥吧，我俩吃一个带一个，就杀进了798。

有很多好玩的，依然照片说话。

超级喜欢这个，想一起合影，WD不乐意。。。哎。。。。

这个像不像留着口水的狗？嘿嘿，辛巴达七海传奇。。。。

WD说，我很2。好吧~~~

俺俩唯一的合影，还是我拍的，太遗憾了。。。。

战利品，从没想过人品能好到这程度

798里好玩东西太多了。基本上绘画，尤其是特艺术的油画，我比较难理解，有的甚至觉得不是美，而是渗得慌。估计作者想表达的情绪太激烈了。

比较能接受的是雕塑，有很多都好看。可惜好看的周围总是有人排着队等拍照，导致我连拍个空镜的机会都没。还有摄影展，比较好懂，也很好看。

这里拍写真的也很多，还有拍婚纱照的。个人觉得这里不适合结婚拍照，太。。。什么了，反正感觉温馨少了点。对，还有遇上一suai锅拍写真，更奇妙的是，我们走到哪，出来就看见他也在门口拍照，哎，sou不了了，太矫情了。还听到摄影师充满耐心的让他体会抽烟的感觉，去摘一朵花之类了。怎么会有这么自恋的娃啊~~~~

对了，还有一个好玩的就是熊猫慢递。据说收费比较贵。本来WD建议我给自己的下一个生日写个明信片。我想了想，有点害怕。自己现在的生活充满的太多的不确定性。虽然没有太多遗憾不满，可是，却不知道要为未来的自己写点什么。现在的状态，太鸡肋了，没什么可抱怨的，可也不值得炫耀。平淡到不知道该说什么。现在练就了把任何琐事放大的本事，吃个好吃的都开心的合不住嘴，其实想想挺没出息的。

还坐下来翻看了大家记录在本子上的文字，很多情绪被激发。恩，我不想影响游玩的心情。也不愿意说什么上次两个人，这次一个人，下次要两个人之类的话，没劲。于是，作罢。怀着复杂的心情离开。我很喜欢这家店的风格，温馨。

又想起一事儿，有一个展览，惨了，名字我竟然说不上来了。。。不过有句话，记得很清楚：一个人太少，两个人太多，所以爱上半个人。我自己背着WD悄悄寻思了半天，哎，啥也不说了。。。。

这里书店也比较多，特有气氛的那种。估计我妹见了会很激动的告诉我，她以后想开的书店就是这样。嘿嘿，以后可以合伙一起来嘛。不过谁来选书很重要啊，我水平不行，可我也不相信您的水平，哈哈。

挺喜欢这地儿的，我俩逛了一整天，出门已经可以吃晚饭了。太累，累的我都没心情讲究吃。随便找了小摊，WD吃了难吃的饭。我吃了难吃的面。PS，我的面比她的饭好吃，娃还来我碗里找吃的。哈哈

这天回家早，吃完饭累惨了，路上又远，俺俩也没再去哪，就回了

到家快9点，去稻香村买了点点心带回家巴结俺爹娘。

over，早早休息。wd晚上抢我被子，哼哼~~~太欺负人

再游北京-2

2nd day

8点半的闹钟，出门九点半，和WD室友顺路，乘坐快速公交。计划去798.

公交上遭贼。一女的，我都没看见过真人，光听见声音了，走在半道上就大吼，不让司机师傅靠站停车，说自己钱包丢了。还迅速报了警。这下好，师傅说公司规定，报警了就不能开门，一车人停在路中间动不了。我们还好，不怎么着急，车上有急着上班的，赶火车的，去医院看病的。。。。。吵成一片。

听了半天，发现，丢钱包的钱包里没钱。。。全是卡。。。。她硬要等警察。。。。哎，大家都无奈了，这一车人呢。。。

中间有个小插曲，差点没笑死我。

那会儿我们仨都找到了座儿，坐那直喊饿。然后，旁边一大叔，年纪顶多跟我爸一样大的大叔，头也不回，撅着屁股就往WD座位上挤。。。。。眼瞅着就坐她腿上了。。。。哈哈哈，这大叔，一声招呼也不打，还自言自语的喊累。。。。吓的娃直接跳起来，客气的说，座位你做吧，我站会。。。。

太可怜了，哈哈哈，受了惊吓的孩子。。。。。

我们总结的经验就是，坐在公交车上的时候，座位一定要坐满了，千万别留下一点空隙。要拿出古时候皇帝坐龙椅的范儿来。。。。太好玩了，娃就这么在我面前站了半小时多

插曲完毕。

继续说遭贼的事儿。

话说报警以后，警察竟然说要半小时才能到。好吧，那就等呗。

那赶火车的啊，一直吼着说自己着急，冲着大伙儿使劲儿抱怨，可是司机师傅叫他商量事儿的时候，他硬是没听见，也不搭理，光顾着自己喊叫着急。。。。看来娃一点也不急。。。。。。。。

还听见一孩子，好像是在百度上班，说迟到了要罚200。主动找到丢钱包的，让她检查了没问题，从司机那里下车了。好玩的是，由于是快速公交，专用车道，娃下车了，也还是困在路上，没办法跨出去打车，爬在栏杆上痴痴地望着。。。。。

然后呢，警察终于来了。穿着霸气，架势十足的，从另外一辆快速公交车上跳下来。大家伙儿都惊了，哈哈哈。太亲民了这也。。。。

来了就来了呗。悄不出出的问了几句话，就不见了。。。丢钱包的也不见了。。。。。

好吧，大家稀里糊涂的坐着车就又出发了。后来有人怀疑，那丢钱包的八成是自己在哪个口袋找着钱包了，没话说了。嘿嘿

经过这事儿一磨机，到前门就快12点了，798太远了，饿得不行，就想赶紧吃饭。。。改计划，去了后海。

恩，路上，应该是在鼓楼附近，吃了北京小吃。面茶，爆肚，奶油什么酥，炸什么芝麻什么酥，还有豌豆黄，都很好吃。吃的着急，人也多，没顾上拍照。

吃完先去了后海，特美，上照片。每一张都很好看啊，贼喜欢这个地方。

旁边有很多胡同，什么烟袋斜街啊，黑芝麻胡同啊，名字都特好玩儿。看见了这只懒洋洋的猫。。。。

吃了很多好吃的。。。。炸灌肠，超好吃，就是一股蒜味儿不太好，照片没传上来。。。

这个叫三角烧，味道一般，像我在西外吃的铜锣烧。之所以买这个，因为。。。就他家门口没人排队。。。。

南锣鼓巷有很多卖奶酪的，人多，我没去。。。。这个港式奶茶很好喝。其实，我出门前自带了水杯，可还是忍不住消费了。

南锣鼓巷口，主要用意是显摆我新买的包。

 

我家门口也这样就好了。。。

烦死了，这相册，一会能传一会不能的。。。。哎，拉到，不能算了。。。

这是在南锣鼓巷。。。。。

这车很霸气啊。。。

就在这么点儿地方，我俩逛了整整一天。。。。好玩的小店太多了，逛的贼开心。还买了一顶小熊的帽子，嘿嘿，装嫩。

晚饭在黄记煌解决，吃了俩小时，就知道吃了多少了。。。。

出来又去后海遛了一圈，晚上也很美。我喜欢这地方。。。。

对，就是这天下午，收到一条短信，说我中奖了。国家大剧院歌剧节的抽奖活动。嘿嘿，那时候还不知道奖品是什么，激动的不行。人品不错哈哈哈

学习LaTeX-2-图片操作

插入图片：

最前面要加上这两句，在begin document之前。\usepackage{CJK}\usepackage{graphicx}需要图片的时候：\begin{figure}\includegraphics{这里写同一个文件夹内的你要编辑的图片名}\caption{这里写图名} \end{figure}调整图片大小：\includegraphics[width=0.8\textwidth]{fig1.png}，这样改就成。textwidth是文字宽度，0.8就是说是文字宽度的0.8倍。个人觉着这样相对的大小比较容易掌握。并列插入两幅图像：\begin{figure}\begin{minipage}[t]{0.5\linewidth}\centering\includegraphics{fig2.jpg}\caption{图名}\end{minipage}\begin{minipage}[t]{0.5\linewidth}\centering\includegraphics{fig3.png}\caption{图名}\end{minipage}\end{figure}这个插入的是两幅不相关的图片，如：fig2，fig3。插入子图的话（如fig4(a)和fig4(b)）也可以。但是我用了下面的方法。插入子图：\begin{figure}  \centering  \subfigure[子图1名]{    \label{fig:subfig:a} %% label for first subfigure    \includegraphics{fig4a.jpg}}  \hspace{1in}  %控制两张图片之间的距离  \subfigure[子图2名]{    \label{fig:subfig:b} %% label for second subfigure    \includegraphics{fig4b.jpg}}  \caption{fig4名字}  \label{fig:subfig} %% label for entire figure\end{figure}图像太大就会竖着排放，可以继续按照上面设置width的方法调整合适的大小，达到并列排放的效果。

学习LaTeX-1

段首缩进：

默认的是首段不缩进，后面的才会有。加上usepackage{indentfirst}就会每段都缩。

还有一个，是我自己一开始就没有注意到的。在LaTeX中，回车并不作为新一段的开始，需要空一行才能识别。

 

关于项目编号：(numerate编号)

假如连续几个编号之间隔了几段正文，或者说，想要从一个非1的编号开始。如：

\begin{enumerate}
\item  AAA
\end{enumerate}

XXX

\begin{enumerate}
\item BBB
\end{enumerate}

可以这样

\begin{enumerate}
\item  AAA
\end{enumerate}

XXX

\begin{enumerate} \stepcounter{enumi}
\item BBB
\end{enumerate}

这样，BBB的编号就是2。假如需要编号为3，4，5 。。。

\begin{enumerate}
\item AAA \item AAA
\end{enumerate}
XXX
\begin{enumerate}\stepcounter{enumi}\stepcounter{enumi}
\item BBB
\end{enumerate}

这样的话，BBB的编号就是从3开始。明白没？也就是说从n开始编号，\begin{enumerate}后面就要写n-1个\stepcounter{enumi}。

			再逛北京-1
	

	

		

                                    
根据我的笔记，4月15日晚上8点半的火车，硬座，次日8点到北京。


由于是周五，WD上班，www去天津出差，ocean学校也不方便，这天我一个人玩儿。


先去了国家大剧院，前一天在网上预订了参观票，省10RMB。




大剧院内部还是很霸气的~~~~




上图是歌剧厅的一个入口。后面某天看歌剧，就在这里，超大。。。


正好这天是大剧院歌剧节的第一天，参观到中午，等到了一场小演出，西藏爱乐乐团。


为啥我听着他们创作的民歌总有种到了云南的感觉。。。。不过后来有几个宫廷乐的演奏，还是很好听的。


看完12点，后面的项目比较晚，饿了。于是出来找了一好心小姑娘帮忙拍照。姑娘太善良了，我都憔悴成啥了，还让她拍的珠圆玉润。。。。




大剧院出来，又去天安门转悠。原本想去参观毛爷爷纪念堂，可是12点那里就关门了，我不知道。。。。


后来再次路过这里，见识了排队领票的人流多么壮观，事实证明，我没去凑热闹是非常正确的。


在天安门广场看到的一幕，感觉很好，差点也冲上去要合影。。。。。看来这个暑假，非得去一趟藏区了。




北京的安检真的非常严格。这个每个地下通道，地铁站口都有安检。而外面远远看着像蒙古包一样的地方，是安检2。很多很多。。。。据说是因为俄罗斯的地铁爆炸事件。。。。




接着，找地儿吃饭。坐地铁去了西直门，崔介绍的好吃泡芙我没找着，随便进了家店，康师傅什么快餐。太太难吃了，一股泡面味儿，还20RMB，吃的我很不爽，也没心情在那商场逛下去。


坐了两站路的公交，去了传说中买衣服的动物园。跟康复路差不多，很大，很乱，人很多，我受不了，走马观花的就又闪人了。


这天的倒数第二站，西单。


别看没去什么地儿，但时间过得还贼快，大城市就是这样，路上太耽误事儿，幸亏有地铁，要不一天全费路上了。


到西单就5，6点了。背着大书包腰都快折了，实在是逛不动啦，坐等WD来接。


大概6点半，终于碰头了。去吃了pizza。不贵，俩人吃了80。一个pizza，一份沙拉，一壶茶，还有鸡翅。之前我在麦当劳垫了点，所以这点东西正好够我俩。


倒车的地方正好在前门，就去前门大街和大栅栏小遛了一圈，也就那样。大栅栏确实有点像咸阳的嘉惠，哈哈。




WD住的很远，晃悠回去，快10点。累惨了，还坐在床边看了会非诚勿扰，这节目确实比较搞笑。哈哈，一天还算充实。一个人瞎转。。。


明天WD带着我，就轻松多了。。。。。。
