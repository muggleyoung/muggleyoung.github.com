群硕软件-笔试

跟腾讯比起来，我觉得前者的题目出得好，考察于无形之中~

比如，问，什么是操作系统？操作系统管理啥？

第一问我就不会。。。

问，什么是软件工程？

不会，后面还有一问，甚至都不记得了

问，什么是虚析构函数，什么是多态？

天晓得为什么要问这么多“什么是”，一个都不会，还不如让写个例程，体现了多态思想，写个析构函数啥的。不敢说全会，但绝对比“什么是”会的多。

为什么要问这么多概念呢，背过了就算好啦。

没有读代码，就一个算各种size的。现在还会一些了，想当初面试中兴的时候，全不会，那是多么的凄惨呀~~~

写代码，二分查找，字符串复制，字符串类的一些声明函数的实现，最后还有一个实现拼图什么的，显然不会，猜测应该是传说中的动态规划吧，回头好好学学呀~~~~

腾讯笔试

今天应该算是腾讯2011校园招聘在西安揭幕了

软件开发第一批笔试

西工大什么东馆，超多人

笔试竟然还签了什么保密协议。。。不过话说回来，谁能把题目记得那么清楚呀，我不能~

总之，大致就是C++，操作系统，网络。重要性，和题目数量也是是这个次序。

题目出的还是不错的，算是基础，但又得真正理解了，才好做。

C++嘛，就是那些指针呀，继承呀，还考了异常，我不会诶。。。。

操作系统和网络都是最基础的，当然死记硬背没用，该不会还是不会

编程题是超大数排序，模拟小游戏，操作系统汇编分析，进程什么的，基本上是考算法思想的

像我被MATLAB深深“毒害”的娃，看什么题目，都忍不住想，用数组实现多方便，干嘛用指针呀。最近好点了，还能想起来递归下，指针下。功夫都是练出来的啊~

没了，题目做完就忘。实验室一个孩子回来还实现了几个题目的代码，告诉我哪个错了哪个对了。恩，有错有对呀，说不上悲惨，但也好不到哪儿去。

算法呀，还是得好好学，有的公司介意你写代码的水平，有的公司更介意的是算法思想。我觉得后者更重要哦。

QuickSort

为毛我死活学不会计算时间复杂度~

太挫败了~

 

Quick Sort 快排：

 

#include <fstream.h>
#include<math.h>

int partition(int A[], int head,int tail)
{
    int key = A[head];
    int i,j,t;
    i = head;
    j = head+1;
    while (j<=tail)
    {
        if (key>A[j])
        {
            i = i+1;
            t = A[j];
            A[j] = A[i];
            A[i] = t;
        }
        j = j+1;
    }
    A[head] = A[i];
    A[i] = key;
    return i;
}

void quicksort(int A[], int head,int tail)
{
    int r;
    if(head<tail)
    {
        r = partition(A,head,tail);
        quicksort(A,head,r);
        quicksort(A,r+1,tail);
    }
}

void main()
{
    ifstream infile;
    ofstream outfile (“result.txt”);
    infile.open(“seq.txt”);
    int n = 0;
    int A[100];
    while (!infile.eof())
    {
        infile>>A[n];
        n++;
    }
    quicksort(A,0,n-1);
    for (int i=0;i<n;i++)
    {
        outfile<<A[i]<<” “;
    }
}

情爱妄想症

 

He loves me,He loves me not

看着这句话，似乎很可爱

一个小姑娘蹲在山顶，手握一朵玫瑰，数着花瓣，他喜欢我，他不喜欢我，他喜欢我，他不喜欢我……

电影本身，可没这么招人怜

前半部讲述一个学艺术的女孩，爱上已婚的心脏科医生，而医生却没有那么爱她，不愿离开自己的妻子。中间姑娘用刀子撕破裙子，拔掉房东心爱的植物，依然有了点可怕的影子

后半部，我看惊悚了。这一切都是幻想，医生全然不知。当这一切大白时，医生愤怒转身，她被“抛弃”，依然砸伤他。她因为妄想症被送入精神病院，他受伤双腿不再灵活。若干年后，她病愈出院，病房衣柜后面发现一个用药片做成的他的画像。。。。。。

这个结局，着实让我从骨子里哆嗦。趁着结尾那句话，真不知道她还能做出什么来。

　　”我爱得疯狂
　　犹幸能用理性平衡
　　我明白要忍耐
　　心存盼望’
　　一名住院50年的色情狂患者这样说”

我。。。我。。。我。。。。

真害怕了~~~

不过想想，爱得疯狂，有时确实一相情愿而已。谁没有点幻想的时候，幻想着他还爱我，实际，人家根本不在乎。何苦呢，是吧~发发牢骚哈~

英文名字挺好，中文名就傻了，天使爱过界。不用塔图演的片子都以天使开头吧，真没意思。法语名字，不认识。。。
