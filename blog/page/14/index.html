BubbleSort

冒泡排序：

#include <fstream>
#include <math.h>
using namespace std;

void BubbleSort(int A[],int n)
{
    for(int j=1;j!=n;j++)
    {
        for (int i=0;i!=(n-j);i++)
        {
            if(A[i]>A[i+1])
            {
                int t = A[i];
                A[i] = A[i+1];
                A[i+1] = t;
            }
        }
    }
}

void main()
{
    ifstream infile;
    ofstream outfile(“result.txt”);
    infile.open(“seq.txt”);
    int A[100];
    int n=0;
    while(!infile.eof())
    {
        infile>>A[n];
        n++;
    }
    BubbleSort(A,n);
    for (int i=0;i!=n;i++)
    {
        outfile<<<” “;
    }
}

网易笔试

网易2011年校园招聘笔试，跟有道和游戏是分开进行的，另外那俩我还没投简历呢~

传说中的笔试题难的bt

我投的测试工程师，一如既往的BT

四部分：计算机基础，Java，数据库，测试

 

计算机基础：没几个题，记得有算法复杂度，tcp三次握手，最后还有一个写代码的题目，我是深深深深的记住了

    整形数组a0,a1,….,an，找出数组中的最大连续子串和，即ax,a(x+1),….,ay的和最大，要求算法复杂度为O(n)，写出思路和代码，不限语言。

   这题纠结了好久，还是把自己绕进去了。。。。。

Java：选择题，读代码题。基本围绕面向对象进行的。类类型，类继承之类。Java我不会，凭印象做的

数据库：忘完了，直接放弃。考了索引类型，写spl语句检索之类

测试：黑盒白盒，测试用例的概念分类什么的；qq能登陆，ie却打不开网页，问你会从哪几个方面解决问题；生产出的一次性水杯，设计测试用例；写一段代码，什么代码忘了，让自己测试该段代码。

 

不同部门不同职位，题目都是不一样的。北京的测试，全考的测试知识；开发，还有智力题要考之类之类。。。

接连几次笔试被鄙视，太太沮丧了，好好看书了我，不投简历先~

腾讯笔试

今天应该算是腾讯2011校园招聘在西安揭幕了

软件开发第一批笔试

西工大什么东馆，超多人

笔试竟然还签了什么保密协议。。。不过话说回来，谁能把题目记得那么清楚呀，我不能~

总之，大致就是C++，操作系统，网络。重要性，和题目数量也是是这个次序。

题目出的还是不错的，算是基础，但又得真正理解了，才好做。

C++嘛，就是那些指针呀，继承呀，还考了异常，我不会诶。。。。

操作系统和网络都是最基础的，当然死记硬背没用，该不会还是不会

编程题是超大数排序，模拟小游戏，操作系统汇编分析，进程什么的，基本上是考算法思想的

像我被MATLAB深深“毒害”的娃，看什么题目，都忍不住想，用数组实现多方便，干嘛用指针呀。最近好点了，还能想起来递归下，指针下。功夫都是练出来的啊~

没了，题目做完就忘。实验室一个孩子回来还实现了几个题目的代码，告诉我哪个错了哪个对了。恩，有错有对呀，说不上悲惨，但也好不到哪儿去。

算法呀，还是得好好学，有的公司介意你写代码的水平，有的公司更介意的是算法思想。我觉得后者更重要哦。

华为2011校园招聘总结

纠结了好几天，终于狠狠地拒了

从一开始，就没想过真的要去

周围人陆续签约的时候，还是心虚呀

优：

1. 可以呆在西安

1.1 7.5k的工资，就算拿到手6k左右，在西安，绝对小白领；

1.2 离家近，朋友多

2. 我申请的销服，技术要求相对低，还要出差为客户服务，用爹的话说，正是我所欠缺的

3. 同学们都说，这里学的东西多，以后跳槽涨得快（这个我不是很认同诶）

劣:

1. 军事化奋斗者作风，风格明显不适合我，不是说不爱奋斗，过犹不及啊

所以说，道理谁都懂，就是舍不得
