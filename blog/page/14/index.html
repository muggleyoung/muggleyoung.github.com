长假碎碎念-la Vie en Rose

虽然没啥心情，想了想，这么长的假期，这么多的碎事，还是絮叨下吧

这一絮叨，就得从上月29号开始

晚上崔同学勾引我出去玩，结果还没吃完饭，娃就因为感冒挂掉，早早就各回各家了。

后来孩子跟我电话说，那晚上，她男朋友本来打算求婚的，在南湖准备了巨型烟火表演，结果因为感冒统统泡汤。我。。。。我听了都那个遗憾呐，本来还有个机会见证下，激动下呢。

订婚总是件好事。1号冒雨回家，我们去庆祝吃饭喝茶ipad，晚上孩子把持不住硬要喝酒。终于终于，咱喝大了一回。走路撞到了酒瓶，自己浑然不知，只是隐约听见身后有动静，扭脸看见一堆人在收拾啥。。。回家的时候，记忆也仅停留于同学们把我送到了楼下，接下来怎样，我也不知道啊~

早上醒来还一惊，以为隐形眼镜忘了摘。还好，睡衣换了眼镜也摘了，娘说我回来还喊饿，吃了点东西才睡。吃货啊吃货，啥时候这事儿忘不了。。。。

当然，酒还是得少喝，过劲儿我就也感冒廖。。。but哈尔滨还挺好喝的呀，也不贵吧

——————————————————————-

工作没定，在家呆着也不踏实，3号就回学校了，上学这么多年，头回如此积极。。。

还抽空进城悠哒一圈，10点出门，12点带着一Subway就回来了。出息。。。。

商场全都是100当做多少多少花的，真烦人，直接打折不行么，算钱太麻烦了

崇拜lh同学呀，当初带着我在万达，用一小坡手机硬是算出最优解，买了一堆东西回来。我一个人是没这耐心了，不如淘宝去

—————————————————————–

放假期间没少投简历，根本都不记得投了什么，可看看网站上，怎么还有那么多公司没有投，真的要疯掉了。

还有很多带open questions的，我都还偷懒没回答呢

昨天生病，挂掉一天在床上睡觉。今早在李同学的鼓励下，硬是把P&G的在线测试搞定了，其实，我真没抱希望，人家能进去，自然是有道理的，只是咱当初没看到人身上的什么特质而已，我希望还是不怎么大呀。

快消，四大这些，先放一放吧，有时间我也投下，没有相应的基础，我也从前没贮备点知识，申请起来，总感觉不靠谱。

早上胖娃电话慰问我，顺便说俊酱也签华为了。额额额，真的至于么，我疯了，俊酱多牛的娃，干嘛不等研究所呀，太可惜了。

压力越来越大鸟~

———————————————————————-

中午整理证件发现怎么都是英语的东西，toeic的，tofel的，乱七八糟的，要是gre能多考几分就好了，看起来有点牛人的样子起码。

恩，某同学，不要骂我，没复习gre我知道错了，低估自己英语应试的本事了。回头心情好了，再考一个去，就不知道考了有毛毛用处

实验室开始限制流量，不能下电视看啦。听歌呗。发现音乐库里已经有至少5个版本的la vie en rose了，真好，多收集点，太有爱啦。尤其是两小无猜里面那个童声版本，可惜我没有。。。。。

BubbleSort

冒泡排序：

#include <fstream>
#include <math.h>
using namespace std;

void BubbleSort(int A[],int n)
{
    for(int j=1;j!=n;j++)
    {
        for (int i=0;i!=(n-j);i++)
        {
            if(A[i]>A[i+1])
            {
                int t = A[i];
                A[i] = A[i+1];
                A[i+1] = t;
            }
        }
    }
}

void main()
{
    ifstream infile;
    ofstream outfile(“result.txt”);
    infile.open(“seq.txt”);
    int A[100];
    int n=0;
    while(!infile.eof())
    {
        infile>>A[n];
        n++;
    }
    BubbleSort(A,n);
    for (int i=0;i!=n;i++)
    {
        outfile<<<” “;
    }
}

网易笔试

网易2011年校园招聘笔试，跟有道和游戏是分开进行的，另外那俩我还没投简历呢~

传说中的笔试题难的bt

我投的测试工程师，一如既往的BT

四部分：计算机基础，Java，数据库，测试

 

计算机基础：没几个题，记得有算法复杂度，tcp三次握手，最后还有一个写代码的题目，我是深深深深的记住了

    整形数组a0,a1,….,an，找出数组中的最大连续子串和，即ax,a(x+1),….,ay的和最大，要求算法复杂度为O(n)，写出思路和代码，不限语言。

   这题纠结了好久，还是把自己绕进去了。。。。。

Java：选择题，读代码题。基本围绕面向对象进行的。类类型，类继承之类。Java我不会，凭印象做的

数据库：忘完了，直接放弃。考了索引类型，写spl语句检索之类

测试：黑盒白盒，测试用例的概念分类什么的；qq能登陆，ie却打不开网页，问你会从哪几个方面解决问题；生产出的一次性水杯，设计测试用例；写一段代码，什么代码忘了，让自己测试该段代码。

 

不同部门不同职位，题目都是不一样的。北京的测试，全考的测试知识；开发，还有智力题要考之类之类。。。

接连几次笔试被鄙视，太太沮丧了，好好看书了我，不投简历先~

群硕软件-笔试

跟腾讯比起来，我觉得前者的题目出得好，考察于无形之中~

比如，问，什么是操作系统？操作系统管理啥？

第一问我就不会。。。

问，什么是软件工程？

不会，后面还有一问，甚至都不记得了

问，什么是虚析构函数，什么是多态？

天晓得为什么要问这么多“什么是”，一个都不会，还不如让写个例程，体现了多态思想，写个析构函数啥的。不敢说全会，但绝对比“什么是”会的多。

为什么要问这么多概念呢，背过了就算好啦。

没有读代码，就一个算各种size的。现在还会一些了，想当初面试中兴的时候，全不会，那是多么的凄惨呀~~~

写代码，二分查找，字符串复制，字符串类的一些声明函数的实现，最后还有一个实现拼图什么的，显然不会，猜测应该是传说中的动态规划吧，回头好好学学呀~~~~

腾讯笔试

今天应该算是腾讯2011校园招聘在西安揭幕了

软件开发第一批笔试

西工大什么东馆，超多人

笔试竟然还签了什么保密协议。。。不过话说回来，谁能把题目记得那么清楚呀，我不能~

总之，大致就是C++，操作系统，网络。重要性，和题目数量也是是这个次序。

题目出的还是不错的，算是基础，但又得真正理解了，才好做。

C++嘛，就是那些指针呀，继承呀，还考了异常，我不会诶。。。。

操作系统和网络都是最基础的，当然死记硬背没用，该不会还是不会

编程题是超大数排序，模拟小游戏，操作系统汇编分析，进程什么的，基本上是考算法思想的

像我被MATLAB深深“毒害”的娃，看什么题目，都忍不住想，用数组实现多方便，干嘛用指针呀。最近好点了，还能想起来递归下，指针下。功夫都是练出来的啊~

没了，题目做完就忘。实验室一个孩子回来还实现了几个题目的代码，告诉我哪个错了哪个对了。恩，有错有对呀，说不上悲惨，但也好不到哪儿去。

算法呀，还是得好好学，有的公司介意你写代码的水平，有的公司更介意的是算法思想。我觉得后者更重要哦。
