滑铁卢大学取得成功并非偶然,校方在办学理念上有许多亮点,其中之一是世界上第一所发起Co-Operative教育计划的学校,也是这一计划涵盖学生数量最多的大学;多伦多和滑铁卢地区的许多诸如RIM、Dalsa、Sirific W ireless、Handshake VR和IntelligentMechatronic Systems等高科技公司都是从滑铁卢大学的实验室演化或由毕业生直接创建的。

Co-Operative教育计划,简称Co-Op计划,最早由滑铁卢大学在1964年设立,至今校内约有70%的本科生注册该计划。所谓Co-Op计划,是在读本科生从一年级开始定期到企业、机构或相关组织进行实习工作的、将课堂教学和实际工作结合的一种教育模式。在这种模式下,一般4年的本科学业,需要5年完成,其中课堂教学3~3·5年,实习工作1·5~2年。

滑铁卢大学每学年分为三个学期,冬季学期(1~4)、(5~8)(9~12月),除国家和校方规定的公共假期,学生没有寒暑假。大多数课程的理论教学集中在秋、冬两学期,实习则安排在春季学期。二者同样以学分评估,注册Co-Op计划者必须5年内至少完成1·5年的实习,每次实习工作至少4个月,从大一开始每年至少要实习一个学期。

不同于国内大学的集中实习, Co-Op计划是一种完全意义上的工作体验。滑铁卢大学专门设有一个Co-Op服务中心,包括完善的网络通讯平台和相关数据库,把在校生和外界有效地联系在一起。需要寻找实习工作的学生从撰写简历、查找工作机会到投递申请、面试、签合同等,和社会上的求职者一样必须经过每一道环节。另一方面,参加Co-Op计划的企业、机构和相关组织也必须在校方提供的平台上发布职位介绍、要求,通过网络接受学生发送的求职申请,一旦确定候选人,也是通过校方提供的远程会议平台直接对学生进行面试。为帮助没有求职经验的低年级学生适应这种“真刀实枪”的求职经历,校方设有专门的辅导教师为学生填写简历提供咨询服务,甚至在每一个面试现场都有教师现场指导学生回答问题。用人单位在经过几轮面试后确定最终人选,而学生也有权利在众多职位中挑选。经过双向选择后学生与用人单位签约,赴该单位工作并获取相应的报酬。在完成一次实习工作后,学生和用人单位相互为对方打分,给出相应评价,其中学生对用人单位的评价会保存在校方的数据库中,供其他学生在挑选用人单位时作为考虑依据。学生通过实习工作为自己挣得大部分学费,每次寻找实习工作,都是一种步入社会的实际演练,随着实习工作的经历逐渐丰富,学生不但学到了书本上学不到的知识,还丰富了人生经历,在学习期间就去过北美、欧洲及亚洲的学生并不少见。这样的毕业生较普通毕业生已经对工作和社会更加了解,也更具有竞争力。而用人单位通过接纳实习生,提前物色优秀人才,通过续签实习工作甚至提前签订毕业后的正式工作作为单位的人才储备。Co-Op计划不仅仅在滑铁卢大学取得成功,在整个加拿大、美国、欧洲和亚洲,已有越来越多的大学和企业加入这项计划。

Co-Op计划不同于国内惯性思维的课间实习,也不同于教师安排妥当的集中实习,更不同于衣食住行均由学校安排到位的生产实习或见习,而是完全独立的实习计划。接纳滑铁卢大学学生实习的单位有加拿大本国、美国、欧洲、亚洲等世界各地的公司,学生一旦被接纳实习,余下的衣食住行完全由学生自己解决,教师完全放手让学生去闯荡世界。

Co-Op计划为学生提供了一个进行实践的真实环境,参加实习工作的学生在单位和正式职员一样按时上下班,参与单位的研发计划、设计并检验产品等,因此滑铁卢大学砍掉了工科学生在校的实验课程,而是将实习工作作为实验计入学分。学生寻找实习机会也很务实,一般低年级的学生因自身缺乏经验,不奢望去大公司或获得较高的实习报酬,只要有一个实习机会就一定想法抓住。而对高年级的学生来说,因自身经历了几轮实习,有了相当的实习经验和体会。对要去的实习公司及报酬待遇要挑剔一些,争取获得最好的实习机会。

滑铁卢大学制定的Co-Op计划安排机动、灵活,有些专业的学生不容易在公司找到实习机会,校方规定,这些学生做些与专业有关的志愿者公益活动也可以作为实习经历。学生一旦取得了实习资格可立即参加实习,而耽搁的理论课程可在下个相应的学期学习。滑铁卢大学的Co-Op计划深得各方赞许,学生在学习理论课程的同时还获得了一定的工作经验,他们步入社会后也深受欢迎,许多人在大三下学期就被大型跨国公司“预订”。

Co-Op计划开始只在工科执行,现在校方对管理、商务、艺术等部分文科专业加入Co-Op计划也作为一种要求。用人单位非常注重有Co-Op经历。一个没有实习工作经历的学生很难找到好工作,甚至找不到工作。所以,学校尽可能创造一切条件使学生参加到实习中去。


说实话，这项目好是好，国内真的能实行么？难~

InsertSort

又开始写数据结构代码啦

多少年一轮回呢

从Insert Sort开始，跟排序杠上了

 

读入文件seq.txt，里面存了好几个整数，具体数目不详；

输出result.txt，写入排序后的数。

算法：insert sort  插入排序

 

#include <fstream.h>
void main()
{
    ifstream infile;
    ofstream outfile (“result.txt”);
    int key;
    infile.open(“seq.txt”);
    int n = 0;
    int A[100];
    while (!infile.eof())
    {
        infile>>A[n];
        n++;
    }
    int i,j;
    for (j=1;j<n;j++)
    {
        key = A[j];
        for (i=j-1;i>=0;i–)
        {
            if (A[i]>key)
            {
                A[i+1] = A[i];
                A[i] = key;
            }
        }
    }
    for (i=1;i<n;i++)
    {
        outfile<<A[i]<<” “;
    }
}

Twilight-月食

不得不客观公正的说说这部片子啦

演员没的说，除了男女猪脚，剩下的又帅又好看~

男的嘛，一开始装酷，觉得还不错，可是，这也不长久啊。长得实在不和我口味，尤其第二部身材明显逊于小狼人。不过演吸血鬼还算合适。

女的嘛，越来越胖啦，演技没提高，完毕。

第三部中，又发现几个养眼的。Jasper，Rosalie。上图中间儿俩。Jasper的演员，不这样更帅。偶然看见他另外的照片，真好看呐~很有味道的，这个发型不好。换了发型的真人真的好看多了~

爸爸妈妈都比猪脚好看。爸爸真的很有古典的感觉，妈妈客串过实习医生格雷里面换脸的那个，eva。Bella的妈妈不如以前好看了，24小时里面的Nina，还记得不？

这个是什么什么fanning，名字忘了，小姑娘才十几岁，表演够老道，俩猪脚惭愧不。。。

以前那个坏女吸血鬼换人了，不好看了，没事，反正已经死掉了。Alice，一直都还不错。看书里面，她应该是和女主角关系比较好，戏份比较多的。

对了，说说书。起初，因为喜欢，想买了原版来看，一直等降价。后来下了电子版看，语言还不错，心情描述呀啥的，很仔细，但又有想象空间。反正就是少女小说似的。but，叙述的角度是我不喜欢的，一直从Bella的角度说，谁谁谁对我怎样好，怎样痴情，怎样喜欢我。到第三部，直接利用人家小狼人。恩，不喜欢~

电影，没了新鲜感。第一部，看着充满时尚感的剪辑，觉得很美很帅。可这都两部过去了，还是那几招，吸血鬼的追逐攻击，没有新花样，狼人本来就笨重，更显不出什么花儿。

音乐，没的说的喜欢。太喜欢。每部中都有好几首我爱的。流行音乐典范呐~

所以说，商业片，不要太苛刻。剧情狗血，可以不关注剧情，看看养眼的演员们，听好听的原声，不失为一种美好的享受。

下一部，我当然继续看。只是不去影院罢廖~
