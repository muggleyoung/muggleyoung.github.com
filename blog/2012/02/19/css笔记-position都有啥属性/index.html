CSS笔记-position都有啥属性

根据css参考手册，是这样解释的

position属性定义元素定位类型

再自己乱解读下下几个属性。

absolute：绝对位置定位，是相对于上一级元素样式的位置定位。

fixed，相对好理解一些。是和absolute有点像，但相对于浏览器窗口的位置进行定位的。会对元素生成一个绝对的位置，元素会一直呆在那个位置，无论周围或者下面又有了什么元素。猜测好多网站让人讨厌的飘窗广告大概就是这么搞得。当然，有些重要的信息，也可以用这种方式显示哈。

relative，是相对于周围元素的位置。

其实吧，absolute和relative的理解还不是很深刻，待体会~

另外一个属性，float就好理解多了。每个float的元素都会按照定义漂移，并且互相不会覆盖。如果页面足够宽，就排排站；不够宽呢，后面的元素就会被挤在下一行显示。
