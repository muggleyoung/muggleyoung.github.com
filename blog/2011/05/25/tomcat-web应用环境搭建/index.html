话说这搭环境啊

mini项目环境也简单

可出点问题就能晕乎乎的折腾好久，比如说某条命令多了一个空格。。。。

 

一个一个说吧～

tomcat：mac下tomcat安装，貌似有dmg文件，也可以从tomcat主页下载zip包。

安装或解压好之后，vi ~/.profile

set tomcat home:后面的目录就是tomcat所在的位置

export TOMCAT_HOME = /Users/twer/workspace/apache-tomcat-7.0.12

一般启动tomcat都要进入bin目录输入startup.sh，钱钱教我可以设置别名

start_tomcat:

alias start_tomcat = $TOMCAT_HOME/bin/startup.sh

stop_tomcat:

alias stop_tomcat = $TOMCAT_HOME/bin/shutdown.sh

这样一来，可以在任何目录下输入start_tomcat, stop_tomcat启动和停止服务器。省事很多

 

ant：

安装ant，在~/.profile中加入：

export ANT_HOME = /Users/twer/workspace/apache-ant-1.8.2

 

git：

安装git，使用git clone 命令clone一个reporsitory，git pull更新代码

 

IntelliJ IDEA：

安装之后，选择project struction->project->project SDK->new->jdk，找到：system-library-java-javaVirtualMachine-1.6.0.jdk

打开自己的工程就ok

哦，好像还得在profile里设置下project home

 

mysql：

安装没啥说的，设置路径

export PATH = /usr/local/mysql-5.5.11-osx10.5-x86/bin$PATH

导入数据库：mysql -u root targetDatabase < sourceDatabase.sql

导出命令用mysqldump，箭头方向不同而已

 

操作数据库时，我们使用了sequel pro，安装设置好连接参数就行

 

然后！start_tomcat之后，咱的程序就run起来啦！
